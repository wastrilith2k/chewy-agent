# Chewy Customer Care — File Reference Index

This index maps each saved help-doc file to the URL it was scraped from.

The crawl started at `https://www.chewy.com/customer-care` and followed every link/button on the main page that carries the class `kib-ingress-card`. The URL for each link was reconstructed from its `aria-describedby` attribute (the `section-` prefix and `-desc` suffix were stripped). Each resulting page often contains several inline article sections covering different sub-topics, all preserved in the saved file.

Note: the user-mentioned class `kip-ingress-card` is actually `kib-ingress-card` (with a "b") in the live DOM.
